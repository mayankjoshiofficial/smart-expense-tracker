# Smart Expense Tracker (Python)

A beginner-friendly Python CLI project to track daily expenses.

## Features
- Add expenses with category
- View all expenses
- Calculate total spending
- File handling with persistent storage
- Error handling for invalid input

## Concepts Used
- Functions
- File handling
- Exception handling
- `os` module
- Python standard library

## Future Improvements
- Date-wise filtering
- Category-wise summary
- Delete/Edit expense
- CSV support
